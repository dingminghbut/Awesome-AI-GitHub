# Contributing

Thanks for helping improve Awesome AI GitHub.

## Suggest a Project

Open a project suggestion issue and include:

- GitHub repository URL
- Best category
- Why the project is useful
- License, if known

Good suggestions help developers, researchers, or founders discover something they can learn from, run, compare, or build with.

## What Fits

- Public open-source repositories
- AI, LLM, Agent, RAG, Generative AI, AI Infrastructure, Evaluation, or Robotics projects
- Projects with clear documentation or active development
- Tools, frameworks, templates, benchmarks, datasets, and practical examples

## What Does Not Fit

- Private repositories
- Pure marketing pages without usable source code
- Spam or low-effort forks
- Projects unrelated to AI

## Ranking Notes

The ranking is automated. Suggestions do not guarantee placement. The updater still uses GitHub metadata, recent star growth, freshness, and category matching.

# 贡献指南

感谢你帮助改进 Awesome AI GitHub。

## 推荐项目

请通过项目推荐 issue 提交：

- GitHub 仓库链接
- 最合适的分类
- 项目为什么有用
- 已知 license

好的推荐应该能帮助开发者、研究者或创业者学习、运行、比较或构建。

## 适合收录

- 公开开源仓库
- AI、LLM、Agent、RAG、生成式 AI、AI 基础设施、评测或机器人相关项目
- 文档清晰或近期活跃的项目
- 工具、框架、模板、基准、数据集和实践示例

## 不适合收录

- 私有仓库
- 没有可用源码的纯商业落地页
- 垃圾内容或低质量 fork
- 与 AI 无关的项目

## 排名说明

榜单由脚本自动生成。推荐项目不保证一定上榜，更新器仍会根据 GitHub 元数据、近期增星、新鲜度和分类匹配进行筛选。
